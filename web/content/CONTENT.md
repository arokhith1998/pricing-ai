## How to publish a post

1. Copy an existing file in `content/blog/`, name it `YYYY-MM-DD-slug.md`.
2. Fill the frontmatter (`title`, `date`, `author`, `excerpt`, `published: true`).
3. Write the body in Markdown. Commit. It goes live on its `date`.
